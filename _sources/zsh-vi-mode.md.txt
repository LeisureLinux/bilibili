### zsh 命令行下的 line editor 
#### zsh zle vi 模式
  - 移动
    - $ 行尾
    - ^ 行首(第一个非空)
    - 0 行首
    - [n]w 前进 n 个单词  
    - [n]W 前进 n 个单词(空格隔开)
    - [n]e 前进到 n 个单词的结尾 
    - [n]E 前进到 n 个单词(空格隔开)的结尾
    - [n]b 后退 n 个单词
    - [n]B 后退 n 个单词(空格隔开)
    - [n]t{char} 向右移到第 n 个字符的左边一个字符
    - [n]T{char} 向左移到第 n 个字符的右边一个字符
    - [n]f{char} 向右移到第 n 个字符上
    - [n]F{char} 向左移到第 n 个字符上
    - [n]; 重复上次的 tTfF 操作 n 次
    - [n], 重复上次的 tTfF 反向操作 n 次
  - 插入/修改/删除
    - [n]iIoOaA （忽略介绍）
    - [n]cw （忽略介绍）
    - [n]dw （忽略介绍）
    - [n]yw （忽略介绍）
  - 围绕
    - va{ 选择花括号内内容(含花括号)，然后用 c/d/y 等动作
    - di' 删除单引号内的内容，不含单引号
    - ci" 修改双引号内的内容，不含引号
    - ya( 复制括号内的内容，含括号
  - 增加/减少
    - Ctl-a 增加数字,日期(月份，星期)，十六进制
    - Ctl-x 减少数字,日期(月份，星期)，十六进制
#### zsh zle emac 模式 (bindkey -e)
  - Ctr-a: 移动到行首
  - Ctr-e: 移动到行尾
  - Ctr-b: 后退一个字符
  - Ctr-f: 前进一个字符 
  - Alt-b: 后退一个单词
  - Alt-f: 前进一个单词
  - Alt-d: 删除当前光标的单词
  - Ctr-w: 剪切光标前的单词
  - Ctr-k: 删除到行尾,送到剪贴板
  - Ctr-u: 从行首删除到当前位置,送到剪贴板
  - Alt-Del 或者 Esc-Ctr-h: 向前删除一个单词
  - Ctr-d 或者 Del: 删除当前光标的字符
  - Ctr-h: 删除光标前的字符(回退键)
  - Ctr-y: 粘贴文本
