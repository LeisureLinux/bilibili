LoongArch 在 Linux 操作系统上的进展

  1. 2022-02-09 binutils 2.38 开始支持 LoongArch - 在支持的系统上运行 file 命令: ELF 64-bit LSB executable, LoongArch, version 1 (SYSV) - 在不支持的系统上运行 file 命令：ELF 64-bit LSB executable, unknown arch 0x102 version 1 (SYSV)

  2. 2022-03-22: GCC12 开始支持龙芯

  3. 2022-06-03: 内核5.19 开始支持龙芯 ，但是仍旧不能引导
